# 💻BetYourRole
## 🚀 소개
### 프로젝트 소개
2024년도 2학기 컴퓨터공학과 진로탐색제 프로젝트 프론트엔드 저장소입니다.

### 주요 기능
조별과제에서 역할을 분배해줍니다.   
포인트 기반 낙찰 알고리즘을 적용하여 참가자들이 자신의 선호도에 맞게 역할에 포인트를 배팅한 후 그에 따라 배정 결과가 결정됩니다.

## 💻 설치 방법
1. 저장소 복제
```bash
git clone [레포지토리 URL]
cd [프로젝트명]
```

2. 애플리케이션 빌드 및 실행
```bash
npm i
npm start
```

* node.js와 npm이 설치된 환경이라고 가정합니다.
